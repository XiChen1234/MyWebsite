# （开发中）我的个人网站——MyWebsite
我要有一个个人网页——这个想法从刚上大学的时候就有了。但是直到现在才开始行动，确实是有些执行力不足了。不过，有句谚语说得好：
> 种一棵树最好的时间是10年，其次就是现在

只要现在开始做了，那就不算晚。终有一日，未来的你会感谢现在踏上征程的自己。

本项目旨在“推销”自己，将那一页简历中塞不下的内容批量挪到这里。同时，后续这个网站也可以成为我各类线上项目的入口（通过末尾的友链），通过这个网站可以快捷链接到我各个项目展示中，方便大家访问查看，为自己的求职和工作增加竞争力（狠狠的卷）。

本项目整体使用静态html页面进行编写，使用原生CSS进行美化、功能基于原生JavaScript进行实现。总共分：**主页介绍**、**基础信息**、**技能水平**、**学历资历**、**作品集**、**联系方式**等主要板块，通过顶栏进行跳转。

未来期望能够正式在线上进行部署，可以在学生服务器中，也可以使用GitHub Pages功能，我个人目前更偏向于后者（0元购）。

# 如何使用
在仓库中直接下载zip并解压，或使用git命令进行仓库克隆

```sh
git clone https://github.com/XiChen1234/MyWebsite.git
```

即可获得源码，纯静态页面，可根据自己需求进行删改和部署。
