# 我的个人网站——MyWebsite

## 介绍
为了暑假和秋招以及未来的职业生涯发展，我想做一个个人网站项目来展示自己，将简历中塞不下的内容挪批量到这里。同时，后续这个网站也可以成为我各类线上项目的入口，通过这个网站可以快捷链接到我各个项目中，方便面试官查看，为自己的求职和工作增加竞争力

本项目整体使用静态html页面进行构建，使用CSS进行美化、功能基于原生JavaScript进行实现。总共分：**主页介绍**、**个人信息**、**技能水平**、**学历资历**、**作品集**、**联系方式**等主要板块，通过顶栏进行跳转。

## 如何使用
在仓库中直接下载zip并解压，或使用git命令进行仓库克隆
```bash
git clone [仓库链接]
```
随后使用vscode打开，使用live server插件生成虚拟服务器，或根据需要部署在nginx等web服务器上，访问指定ip+端口即可

## 未来展望
未来该项目将会继续进行横向的扩展功能，目前所期望实现的技术点有：
- 响应式布局，实现手机、平板上的优质体验

- skill栏目的展开可以有更丝滑的动态效果
- Portfolio链接的地址可以真正将多个项目连接在一起
- 右上角的翻译功能实现
- 增加明暗风格切换功能
